# HTR External Links Extractor - نسخه 3.0

افزونه‌ای قدرتمند و ماژولار برای استخراج و مدیریت لینک‌های خارجی در سایت‌های وردپرسی.

## ✨ ویژگی‌های نسخه 3.0

- **داشبورد مدیریت دو تب:** تفکیک لیست لینک‌های استخراج‌شده و سیستم گزارش‌گیری لاگ‌ها.
- **جستجو و فیلتر زنده (Live):** جستجوی آنی در متن لینک‌ها (Anchor Text) و آدرس‌ها با استفاده از Ajax.
- **مرتب‌سازی هوشمند:** امکان تغییر چینش نتایج بر اساس تاریخ به صورت صعودی و نزولی.
- **فیلتر آدرس منبع:** امکان محدود کردن نتایج استخراج‌شده به یک صفحه یا آدرس خاص.
- **سیستم لاگ‌گیری پیشرفته:** ثبت و نگهداری تمامی رویدادها، هشدارها و خطاها در دیتابیس برای عیب‌یابی آسان‌تر.
- **معماری ماژولار و استاندارد:** پیاده‌سازی بر اساس الگوهای Singleton و Repository جهت توسعه‌پذیری و پایداری بالا.
- **استخراج از منابع مختلف:** پشتیبانی از پست‌ها، برگه‌ها، محصولات و دسته‌بندی‌های ووکامرس، و پست‌تایپ‌های سفارشی.

## � ساختار پروژه

پروژه با رعایت اصل Separation of Concerns به بخش‌های زیر تقسیم شده است:

- `htr-external-links.php`: فایل اصلی (Bootstrap) و راه‌انداز افزونه.
- `includes/class-htr-el-repository.php`: لایه ارتباط با دیتابیس (مدیریت جداول، لاگ‌ها و کوئری‌ها).
- `includes/class-htr-el-extractor.php`: لایه منطق استخراج لینک‌ها (Business Logic).
- `includes/class-htr-el-dashboard.php`: لایه رابط کاربری ادمین (رندرینگ تب‌ها، جداول و فرم‌ها).
- `assets/`: شامل استایل‌ها (CSS) و اسکریپت‌ها (JS) برای عملکرد زنده و ظاهر واکنش‌گرا.

## � نصب و استفاده

1. پوشه افزونه را در مسیر `wp-content/plugins/` سرور خود آپلود کنید.
2. افزونه **HTR External Links Extractor** را از پنل مدیریت وردپرس فعال کنید.
3. از منوی کناری به بخش **"لینک‌های خارجی"** بروید.
4. برای شروع فرآیند استخراج، روی دکمه **"🔄 اسکن مجدد"** کلیک کنید.
5. نتایج در تب "لینک‌ها" و گزارش وضعیت در تب "گزارش لاگ‌ها" قابل مشاهده خواهد بود.

## 🔧 مشخصات فنی

- **نسخه PHP مورد نیاز:** 7.4 و بالاتر
- **نسخه وردپرس مورد نیاز:** 5.0 و بالاتر
- **جداول دیتابیس:**
  - `wp_htr_el_external_links` (ذخیره لینک‌های استخراج‌شده)
  - `wp_htr_el_stats` (آمار کلی اسکن)
  - `wp_htr_el_logs` (ثبت گزارش‌ها و خطاها)
